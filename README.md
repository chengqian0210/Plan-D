# plan D

这是一个关于飞行器的MATLAB & Simulink仿真。
它可以通过链接flightfear提供可视化的仿真结果，借助Simulink Coder可以提高开发速度
当然，一切的前提是你已经得到了对象的数学模型。本项目不提供有关系统建模、辨识的方法，
而仅展现一个可行的仿真方案。

## Getting Started

本项目在Windows下借助MATLAB进行开发，要运行这个工程，首先要了解MATLAB的Simulink Aerospace blockset、matlab Project Management
以及git和Simulink 工程中的源代码管理。值得一提的是Simulink 工程使用git是非常方便的，正常情况下它完全不需要命令行而是
通过图形化界面就可以操作。

### Prerequisites

需要的软件：

1.MATLAB2018及以上版本。
2.flightgear。
3.git。

### Installing

克隆本工程到本地之后你可以按照如下步骤启动该项目
1.打开MATLAB,并将文件夹定位到工程所在目录
2.双击工程根目录的PRJ文件PlanD，即可开启仿真程序
